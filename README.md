# 🤖 AI Chatbot with GUI

An advanced Python-based AI chatbot with a graphical user interface.

## 🚀 Features
- Chat-based interaction
- GUI using Tkinter
- Beginner-friendly project
- Easy to expand with APIs

## 🛠️ Tech Used
- Python
- Tkinter

## 📂 How to Run
1. Download the project
2. Open terminal
3. Run:
   python chatbot.py
